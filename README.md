## Install

```bash
npm i -g lls-cli
```

**Requirement**: Node version >= 16.0.0

## Usage

Execute `lls` in the project root directory, you will see node scripts list

```bash
$ lls

? select node script › - Use arrow-keys. Return to submit.
❯   fmt - prettier --write .
    build
```
